## ✨ Overview

**Saaz** is a modern social music platform where people can listen together, create shared rooms, chat in real time, and connect with others who share similar music tastes. Whether you're enjoying music alone or hosting a listening session with friends, Saaz transforms music into a shared experience.

Built with **React, TypeScript, Tailwind CSS, and Lovable AI**, it delivers a fast, responsive, and intuitive user experience.

## 🌟 Features

- 🎧 **Listen Together** – Create rooms and enjoy music with friends in real time.
- 🎵 **Solo Mode** – Listen to your favorite music anytime.
- 🤝 **Music Match** – Discover people with similar music tastes.
- 💬 **Live Chat** – Chat with friends while listening together.
- 🚀 **Instant Room Sharing** – Invite others and start jamming instantly.
- 📱 **Responsive Design** – Optimized for desktop, tablet, and mobile.
- ⚡ 🎨 Modern & Intuitive UI* – Built with React, TypeScript, Tailwind CSS.
